```
初始化
git init
git add README.md
git commit -m first commit
git branch -M main
git remote add origin https://github.com/sweeteee/Phase 4 - Machine Learning and Multi-Scenario Project Practice.git
git push -u origin main
-- 把本地的 main 分支和远程的 origin/main 分支关联起来；
-- 以后你就能直接用 git push 或 git pull 自动同步。
git pull
-- 是Git 中用于从远程仓库获取(fetch)最新的更改,并与本地分支合并(merge)的命令。
```
